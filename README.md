# Pure Dev Dashboard

A comprehensive developer productivity dashboard and Chrome extension. Features include a Pomodoro timer, Kanban board, music player, code snippets, financial tracking, and wellness tools to simplify your workflow.

## Features

- **Pomodoro Timer**: Stay focused with customizable work/break intervals.
- **Kanban Board**: Manage your tasks visually.
- **Music Player**: Lo-fi, meditation, and white noise for deep work.
- **Code Snippets**: Save and reuse your favorite code blocks.
- **Finance**: Track your earnings and expenses.
- **Wellness**: Breathing exercises and breaks reminders.
- **GitHub Integration**: View your contributions and activity.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How to Run Locally

Prerequisites: Node.js & npm installed.

```sh
# Step 1: Clone the repository
git clone https://github.com/LuizPauloDesigner/pure-dev-hub-center.git

# Step 2: Navigate to the project directory
cd pure-dev-hub-center

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Building for Production

To create a production build (or Chrome Extension build):

```sh
npm run build
```

The output will be in the `dist` directory.
